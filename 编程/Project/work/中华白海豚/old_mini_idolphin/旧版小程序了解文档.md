# 其他信息

index（视频播放页面，首页）、home（主页）、list（菜单选择页）、rankShare（勋章分享页）和end（结尾页）这几个页面较为简单。



picture（白海豚博物馆）、topic（白海豚小课堂）和 activity（线下登记）这几个页面较为复杂，与组件相互获取对方实例，互相调用对方方法，耦合性极强，维护难度大。
